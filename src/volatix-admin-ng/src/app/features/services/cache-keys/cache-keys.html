About Cache Keys

Cache keys are unique identifiers for cached data stored by the agent. Each key corresponds to a specific set of parameters and their cached response.

Available Actions

  • Search - Filter cache keys by typing in the search box
  • View - Inspect the cached data for a specific key
  • Remove - Delete a cache entry permanently
  • Copy - Copy cache data to clipboard (in detail modal)
